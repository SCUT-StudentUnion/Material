参与者
借用人，仓库值班人员
主成功场景
  1. 借用人携带需要归还的物资至学生会仓库
  2. 仓库值班人员登录系统
  3. 仓库值班人员选择对应的借出申请
  4. 录入：仓库值班人员录入归还的物资
  5. 保存：仓库值班人员保存归还记录
扩展场景
  • 录入
    1. 点击全部归还以归还该申请借出的全部物资
    2. 在某一物资种类中点击点击全部归还以归还该申请借出的全部该种类的物资
  • 保存
    1. 当要归还的物资编号已在库中，或未在借出申请中，或归还物资数量超过借出数量时，系统显示提示，仓库值班人员可选回到录入
备注
  • 录入时，也需要视情况决定录入编号或数量，具体参见借出物资
