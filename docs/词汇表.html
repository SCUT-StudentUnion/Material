--- title: 词汇表 ---

{{ page.title }}

User
物资管理系统的用户，如学生会办公室和仓库的值班人员，团委的老师等。
Organization
借用物资的组织。
Person
前来借用物资的人。一个人可以属于0个或1个组织。
Material Category
物资分类，例如：桌子、帐篷。
Use Application
物资申请借出记录。
Applicant
物资借出的申请人。这是一个Person
Return Record
物资借出的归还记录。同一个借出申请可以对应多个归还记录。
Inventory
当前的库存物资记录。
Inventory Check
库存清点情况记录。库存清点也可以用于从异常的状态中回复，可以用它来手动调整当前库存数据。
Counted Record
没有编号，只记数量的物资借出/归还/库存/库存清点记录。
Numbered Record
记录物资编号的物资借出/归还/库存/库存清点记录。
